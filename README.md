# BMI Calculator Web Application (Flask)

This project is a simple Body Mass Index (BMI) Calculator developed using Python and Flask as part of my internship at Oasis Infobyte.  
The application allows users to enter their height and weight and calculates their BMI instantly.

## âš™ How the Project Works

The user enters height and weight in the web page.  
The values are sent to the Flask backend using routes.  
Flask processes the data and applies the BMI formula:

BMI = weight / (height * height)

The calculated BMI is returned to the HTML page and displayed to the user.
